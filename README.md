# BELLYSHOP ULTIMATE FINAL

이 파일이 최종 운영본입니다.

## 교체 방법
1. GitHub 저장소의 기존 파일을 모두 삭제합니다.
2. 이 ZIP을 압축 해제합니다.
3. 아래 5개 파일을 GitHub 저장소 최상단에 전부 업로드합니다.
   - index.html
   - style.css
   - app.js
   - firestore.rules
   - README.md
4. Firebase 콘솔 → Firestore → 규칙에서 기존 내용을 전부 삭제합니다.
5. firestore.rules 내용을 붙여넣고 게시합니다.
6. GitHub Pages 배포가 끝난 뒤 Ctrl+F5로 새로고침합니다.

## 관리자 로그인
- 아이디: admin
- 비밀번호: Firebase Authentication에서 만든 비밀번호

## 포함 기능
- 관리자 로그인
- 상품 등록/수정/삭제
- 신상품/인기상품/추천상품
- 품절/공개/비공개
- 카테고리 자동 생성 및 검색
- 대표 이미지/추가 이미지 URL
- 메인 배너 등록/수정/삭제/자동 슬라이드
- 견적함
- 카카오톡/전화/이메일 상담
- 모바일/PC 반응형
- 사이트 상호명/상담 정보/하단 문구 설정


## 카테고리 관리
관리자 로그인 → `카테고리 관리`에서 카테고리 추가, 수정, 삭제가 가능합니다.

- 카테고리 이름을 수정하면 해당 카테고리를 사용 중인 상품도 자동 변경됩니다.
- 상품이 사용 중인 카테고리는 먼저 상품 카테고리를 변경해야 삭제할 수 있습니다.
- 상품 등록 화면에서는 등록된 카테고리를 선택합니다.
